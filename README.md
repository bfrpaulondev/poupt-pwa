# PoupPT PWA

Progressive Web App de gestao financeira inteligente para portugueses.

## Stack

- React 18 + Vite
- TailwindCSS 4
- Zustand (state management)
- Recharts (charts)
- Lucide React (icons)
- vite-plugin-pwa

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Funcionalidades Fase 1

- Landing page
- Registo/Login
- Onboarding (5 passos)
- Dashboard adaptado ao modo de vida
- 6 Frascos com percentagens editaveis
- AI Coach com personalizacao
- Gestao de dividas (formal + informal)
- Metas financeiras
- Perfil e gamificacao
- PoupMoedas
- Modo Sobrevivencia
- Investimentos
- Configuracoes
